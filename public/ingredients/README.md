# Ingredient Images

This folder contains pixel art images for each ingredient.

## Image Requirements

- **Format**: PNG (preferred) or SVG
- **Style**: Chibi/Pokemon-style pixel art
- **Size**: 200x200px recommended
- **Naming**: Use the ingredient ID (e.g., `dungeness-crab.png`)

## Current Placeholders

The `.svg` files are temporary placeholders with emojis. Replace them with your pixel art PNG files:

- `dungeness-crab.png` (or .svg)
- `rock-greenling.png` (or .svg)

## Adding New Ingredients

1. Create your pixel art image (200x200px PNG)
2. Name it with the ingredient ID (lowercase, hyphenated)
3. Save it in this folder
4. Update `/data/ingredients.ts` with the image path
